E-Repair Shop UI Design

Project information

  • Category : Figma - UI/UX
  • Project Live Preview

As a dedicated mobile app designer, I successfully crafted the user interface for an E-Repair shop mobile application for mu client. From the ground up, I transformed detailed documentation into an intuitive and visually appealing design. This comprehensive mobile app caters to three key users: administrators, customers, and technicians. My role involved creating a seamless and efficient experience for each user type, ensuring optimal functionality and a modern aesthetic. Through my design process, I aimed to enhance user engagement and streamline interactions within the E-Repair shop ecosystem.
